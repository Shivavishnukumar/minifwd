Directory Structure

Before we explore the commands used to manipulate the Linux environment, we should take a quick look at the structure of the environment itself. Microsoft Windows users will find the Linux file system to be a familiar structure because it is basically the same. You can think of a Linux file system as an upside-down tree. See the diagram to the right. Okay, so it doesn't look a lot like a tree, but try to keep an open mind. In the right diagram /(Root) contains 3 directories and Linux_Interactive contains 3 files. In left diagram we could see the directories in Terminal.
